# Finance-Automation-App
A personal finance automation tool that will allow users to upload their bank statements and organize their finances based on categories, date, and total expenses.

## Getting Started

* Download or clone the repository to a code editor
* Install the necessary libraries
  ```
  pip install streamlit pandas plotly numpy
  ```
* Run the script
  ```
  streamlit run main.py
  ```

### Libraries

* Pandas
* Plotly
* Stremlit
